## 上篇 · 复变函数论

### 第1章 复变函数

#### 解析函数
- 定义：区域D内每一点可导 → 解析；点态可导 ≠ 点态解析（需邻域可导）
- C-R方程：$\partial u/\partial x = \partial v/\partial y$，$\partial u/\partial y = -\partial v/\partial x$
- 充要条件：u,v可微 + C-R成立；极坐标形式：$\partial u/\partial r = (1/r)\partial v/\partial\theta$
- 初等函数：$e^z$（周期$2\pi i$）、$\sin z$（模无界）、$\ln z$（多值）、$z^\alpha$（多值性）
- 调和函数：$\nabla^2 u = 0$，$\nabla^2 v = 0$；u,v互为共轭调和函数

#### 核心性质
- 解析函数无限可导；$|f(z)|$极值在边界；唯一性定理；Liouville定理（有界整函数为常数）

#### 补充专题：实解析性与复解析性的区别
- **实解析（Real Analytic）**：$f(x)$在$x_0$处实解析 ⇔ 可展开为收敛幂级数 $f(x)=\sum a_n(x-x_0)^n$，收敛半径$R>0$
  - 注意：$C^\infty \neq$ 实解析。反例：$e^{-1/x^2}$在$x=0$处$C^\infty$但不解析（泰勒级数全为零，不收敛到自身）
  - 充分必要条件：泰勒余项趋于零
- **复解析（Complex Analytic / Holomorphic）**：$f(z)$在区域$D$内解析 ⇔ 在$D$内每一点复可导
  - 判定：C-R方程 + 偏导数连续（充要条件）
  - 比实解析严格得多：复解析 ⇒ $C^\infty$（自动）；复解析 ⇒ 泰勒级数收敛到自身；Liouville定理；唯一性定理
- **核心关系**：实解析函数可唯一延拓为复解析函数（限制在实轴上即为原实函数）；复解析函数限制在实轴上得到实解析函数
- **物理应用（S矩阵理论 / 散射理论）**：
  - 物理散射振幅在实能量轴上是实解析的边界值
  - 可唯一解析延拓到复能量（动量）平面 → 极点对应束缚态，割线对应多粒子阈
  - 解析性比动力学方程更基本——用复平面上的解析结构约束可观测量（色散关系）

### 第2章 复变函数的积分

#### 柯西定理
- 单连通域：f解析 → $\oint_C f(z)dz = 0$；积分与路径无关
- 多连通域：复合闭路定理，内边界积分之和
- 柯西积分公式：$f(z_0) = \frac{1}{2\pi i}\oint_C \frac{f(z)}{z-z_0}dz$
- 高阶导数：$f^{(n)}(z_0) = \frac{n!}{2\pi i}\oint_C \frac{f(z)}{(z-z_0)^{n+1}}dz$（解析函数无穷可导）
- Morera定理：逆定理，连续+闭路积分为零 → 解析

### 第3章 幂级数展开

#### 泰勒级数
- $f(z) = \sum a_n(z-z_0)^n$，$a_n = f^{(n)}(z_0)/n!$；收敛半径R = 到最近奇点距离
- 常用展开：$e^z, \sin z, \cos z, \ln(1+z), (1+z)^\alpha$

#### 洛朗级数
- $f(z) = \sum_{n=-\infty}^{+\infty} a_n(z-z_0)^n$；圆环域 $r < |z-z_0| < R$
- 孤立奇点分类：可去（无负幂）、极点（有限负幂）、本性（无穷负幂）

### 第4章 留数定理

#### 留数计算
- 单极点：$\text{Res}(f,z_0) = \lim_{z\to z_0}(z-z_0)f(z)$
- m阶极点：$\text{Res}(f,z_0) = \frac{1}{(m-1)!}\lim_{z\to z_0}\frac{d^{m-1}}{dz^{m-1}}[(z-z_0)^m f(z)]$
- 本性奇点：展开洛朗级数取 $a_{-1}$

#### 留数定理
- $\oint_C f(z)dz = 2\pi i \sum \text{Res}(f,z_k)$
- 实积分应用：三角有理式、无穷积分、含三角函数的无穷积分（Jordan引理）

### 第5章 傅里叶变换

#### 变换对
- $F(\omega) = \int_{-\infty}^{+\infty} f(x)e^{-i\omega x}dx$；$f(x) = \frac{1}{2\pi}\int_{-\infty}^{+\infty} F(\omega)e^{i\omega x}d\omega$
- 性质：线性、位移、相似、微分（$f' \leftrightarrow i\omega F$）、卷积（$f*g \leftrightarrow F\cdot G$）
- $\delta(x) \leftrightarrow 1$；$1 \leftrightarrow 2\pi\delta(\omega)$

### 第6章 拉普拉斯变换

#### 定义与性质
- $\bar{f}(p) = \int_0^{+\infty} f(t)e^{-pt}dt$；$p = \sigma + i\omega$
- 微分：$f'(t) \leftrightarrow p\bar{f}(p) - f(0)$；自动包含初值
- 卷积：$f_1*f_2 \leftrightarrow \bar{f}_1 \cdot \bar{f}_2$
- 反演：留数法、查表法；应用：ODE、电路、振动

## 下篇 · 数学物理方程

### 第7章 数学物理定解问题

#### 三类典型方程
- 波动方程（双曲型）：$\partial^2 u/\partial t^2 = a^2\nabla^2 u$；弦振动、声波、电磁波
- 热传导方程（抛物型）：$\partial u/\partial t = a^2\nabla^2 u$；热传导、扩散
- 泊松/拉普拉斯方程（椭圆型）：$\nabla^2 u = f$（或=0）；静电场、稳态温度

#### 定解条件
- 初始条件：$u|_{t=0} = \varphi(x)$；波动方程还需 $u_t|_{t=0} = \psi(x)$
- 边界条件：Dirichlet（第一类）、Neumann（第二类）、Robin（第三类）
- 方程分类：$\Delta = B^2 - AC$；>0双曲型，=0抛物型，<0椭圆型

#### 达朗贝尔公式
- $u(x,t) = \frac{1}{2}[\varphi(x+at) + \varphi(x-at)] + \frac{1}{2a}\int_{x-at}^{x+at}\psi(s)ds$
- 物理意义：初始位移分解为左右传播的两列波，速度为a

### 第8章 分离变数法

#### 核心思想
- $u(x,t) = X(x)T(t)$；将PDE转化为ODE组
- 求解五步骤：①分离变量 ②求本征值问题 ③求时间解 ④叠加 ⑤定系数
- 适用范围：齐次方程+齐次边界条件、有限区域、规则边界
- 物理图像：驻波解，每一项对应一种驻波模式

### 第9章 二阶常微分方程级数解法 · 本征值问题

#### S-L型方程
- $\frac{d}{dx}[k(x)\frac{dy}{dx}] - q(x)y + \lambda\rho(x)y = 0$
- 核心性质：本征值非负实数、本征函数带权正交、完备性
- 广义傅里叶级数：$f(x) = \sum c_n y_n(x)$，$c_n = \frac{\int f y_n \rho dx}{\int y_n^2 \rho dx}$

#### 与特殊函数衔接
- 勒让德方程：$k=1-x^2, q=0, \rho=1$ → 解为$P_l(x)$
- 贝塞尔方程：$k=x, q=\nu^2/x, \rho=x$ → 解为$J_\nu(x)$

### 第10章 球函数

#### 勒让德多项式
- 方程：$(1-x^2)y'' - 2xy' + l(l+1)y = 0$；$x\in[-1,1]$, $l=0,1,2,...$
- Rodrigue公式：$P_l(x) = \frac{1}{2^l l!}\frac{d^l}{dx^l}(x^2-1)^l$
- 正交性：$\int_{-1}^{1} P_l P_{l'}dx = \frac{2}{2l+1}\delta_{ll'}$
- 母函数：$\frac{1}{\sqrt{1-2xt+t^2}} = \sum_{l=0}^{\infty} P_l(x)t^l$

#### 球谐函数
- $Y_l^m(\theta,\varphi) = P_l^m(\cos\theta)e^{im\varphi}$；$l=0,1,2,...$；$m=-l,...,+l$
- 应用：氢原子薛定谔方程角度解、多极辐射、静电势球坐标展开

### 第11章 柱函数

#### 贝塞尔方程
- $x^2y'' + xy' + (x^2-\nu^2)y = 0$

#### 三类柱函数
- $J_\nu(x)$：第一类，$x=0$处有限
- $Y_\nu(x)$：第二类（诺依曼），$x=0$处发散
- $H_\nu^{(1)} = J_\nu + iY_\nu$，$H_\nu^{(2)} = J_\nu - iY_\nu$：汉克尔函数，描述outgoing/incoming波

#### 关键性质
- 递推：$J_{\nu-1} + J_{\nu+1} = \frac{2\nu}{x}J_\nu$
- 正交：$\int_0^a J_\nu(k_n\rho)J_\nu(k_m\rho)\rho d\rho = 0$（$n\neq m$）
- 渐近：$x\to\infty$时 $J_\nu \sim \sqrt{\frac{2}{\pi x}}\cos(x-\frac{\nu\pi}{2}-\frac{\pi}{4})$
- 应用：圆膜振动、圆柱波导、光纤模式、二维圆形势阱

### 第12章 格林函数法

#### 定义与核心概念
- $\nabla^2 G(\mathbf{r},\mathbf{r}') = \delta(\mathbf{r}-\mathbf{r}')$
- 点源响应：格林函数描述位于$\mathbf{r}'$的点源在$\mathbf{r}$处产生的场
- 叠加原理：连续分布源 = 各点源响应的积分叠加
- 互易性：$G(\mathbf{r},\mathbf{r}') = G(\mathbf{r}',\mathbf{r})$

#### 求解方法
- 电像法：半空间、球域等简单几何
- 本征函数展开：$G(\mathbf{r},\mathbf{r}') = \sum_n \frac{\psi_n(\mathbf{r})\psi_n(\mathbf{r}')}{\lambda_n}$
- 傅里叶变换法：无界区域

#### 基本积分公式
- $u(\mathbf{r}) = \int G(\mathbf{r},\mathbf{r}')f(\mathbf{r}')dV' + \text{边界项}$
- 应用：静电学（点电荷电势）、量子力学（传播子）、声学、电磁学

### 第13章 积分变换法

#### 傅里叶变换法
- 适用于无界域；$\partial^2/\partial x^2 \to -\omega^2$；微分→代数

#### 拉普拉斯变换法
- 适用于含初始条件的演化问题；$\partial/\partial t \to p$；自动包含初值

#### 方法互补性
- 分离变量法：有限区域+齐次边界
- 积分变换法：无界/半无界区域
- 行波法：无界域波动方程
- 格林函数法：非齐次方程

### 第14章 保角变换法

#### 基本性质
- $w=f(z)$在$f'(z_0)\neq 0$处保角：保持夹角大小与方向
- 保圆：圆周/直线变为圆周/直线；保对称点

#### 常用变换
- 分式线性：$w = \frac{az+b}{cz+d}$（最通用）
- 幂函数：$w=z^n$（角域→半平面）
- 指数：$w=e^z$（带域→角域）
- 对数：$w=\ln z$（角域→带域）

#### 应用
- 将复杂边界区域转化为规则区域；求解二维拉普拉斯方程边值问题；静电场、平面无旋流

### 第15章 非线性数学物理问题简介

#### 孤立子（Soliton）
- KdV方程：$\partial u/\partial t + \alpha u\partial u/\partial x + \beta\partial^3 u/\partial x^3 = 0$
- 行波解：$u(x,t) = -\frac{c}{2}\text{sech}^2[\frac{\sqrt{c}}{2}(x-ct)]$
- 正弦-戈尔登方程、非线性薛定谔方程

#### 混沌（Chaos）
- 确定性非线性系统中出现的貌似随机运动；对初值极端敏感
- 典型系统：洛伦兹系统、逻辑斯蒂映射、受驱非线性振子

## 常见学生错误清单

| 错误 | 正确 |
|------|------|
| C-R方程成立就可导 | 需偏导数连续+C-R成立 |
| 点态可导=点态解析 | 解析需邻域可导 |
| 复三角函数有界 | $\|\sin z\|$可任意大 |
| $\ln z$单值 | 多值函数，需分支切割 |
| $\|f(z)\|$有内部极大值 | 极值在边界（极值原理） |
| 有界整函数必有零点 | Liouville：有界整函数为常数 |
| 波动方程和热方程解法相同 | 方程类型不同，解法不同 |
| 分离变量法适用于所有PDE | 需齐次方程+齐次边界+规则区域 |
| 贝塞尔函数和勒让德函数相同 | 不同方程、不同性质、不同应用 |
| 格林函数只用于静电学 | 统一方法，适用于波动/热传导/量子 |
