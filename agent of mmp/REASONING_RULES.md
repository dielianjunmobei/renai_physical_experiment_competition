## 规则1：判定函数是否解析（复变函数第1章）

**适用问题**：判断 f(z) = ... 是否解析

**推理链**：

```
输入：f(z) = u(x,y) + iv(x,y)

Step 1: 分离实部和虚部
        u(x,y) = ...
        v(x,y) = ...

Step 2: 计算四个偏导数
        ∂u/∂x, ∂u/∂y, ∂v/∂x, ∂v/∂y

Step 3: 检查连续性
        四个偏导数是否连续？

Step 4: 验证 C-R 方程
        ∂u/∂x =? ∂v/∂y
        ∂u/∂y =? -∂v/∂x

Step 5: 结论
        若连续+C-R成立 → 解析区域
        若仅某点成立 → 该点可导但不解析
        若不成立 → 不解析
```

## 规则1B：区分实解析性与复解析性（第1章补充专题）

**适用问题**："实解析和复解析有什么区别？""为什么C^∞不一定是解析的？"

**推理链**：

```
Step 1: 明确讨论域
        → 实变量 x ∈ ℝ：讨论实解析性
        → 复变量 z ∈ ℂ：讨论复解析性（全纯性）

Step 2: 实解析判定
        → f(x)在x₀处可否展开为收敛幂级数？
        → 泰勒余项 R_N → 0？（充要条件）
        → 反例：e^(-1/x²)在x=0处C^∞但不解析
        
Step 3: 复解析判定
        → C-R方程成立？（必要条件）
        → u,v偏导数连续 + C-R成立？（充要条件）
        → ∂f/∂z̄ = 0？（等价条件）

Step 4: 对比总结
        实解析：幂级数展开 + 余项→0
        复解析：复可微 ⇔ C-R ⇔ ∂/∂z̄ = 0
        复解析 ⟹ 实解析（限制在实轴），反之不成立
        
Step 5: 物理联系（散射理论）
        → S矩阵元是实能量轴上的实解析函数
        → 可解析延拓到复能量平面
        → 复平面上的奇点结构编码了束缚态、共振、阈值
```

## 规则2：已知实部（或虚部）求解析函数（第1章）

**适用问题**：已知 u(x,y) = ...，求解析函数 f(z)

**推理链**：

```
方法A：偏积分法（利用 C-R 方程）

Step 1: 由 ∂v/∂y = ∂u/∂x → v = ∫(∂u/∂x)dy + φ(x)
Step 2: 由 ∂v/∂x = -∂u/∂y 确定 φ'(x)
Step 3: 积分求 φ(x)，含待定常数 C
Step 4: 组合 f(z) = u + iv，验证

方法B：复数代入法（更快捷）

Step 1: 令 y = 0，得到 u(x,0)
Step 2: 将 x → z：f(z) = u(z,0) + iC
```

## 规则3：复积分计算（第2章）

**适用问题**：计算 ∮_C f(z)dz

**推理链**：

```
Step 1: 判断被积函数 f(z) 在区域 D 内是否解析

Step 2: 选择方法
        → 若解析 + 单连通 → 柯西定理 → 积分为0
        → 若含奇点 → 柯西积分公式/高阶导数公式
        → 若多连通 → 复合闭路定理

Step 3: 识别奇点位置，判断是否在积分路径内部

Step 4: 应用公式计算
        柯西积分公式：f(z₀) = (1/2πi)∮ f(z)/(z-z₀) dz
        高阶导数：f⁽ⁿ⁾(z₀) = (n!/2πi)∮ f(z)/(z-z₀)ⁿ⁺¹ dz

Step 5: 验证结果合理性（量纲、极限情况）
```

## 规则4：幂级数展开（第3章）

**适用问题**：将 f(z) 展开为泰勒/洛朗级数

**推理链**：

```
Step 1: 确定展开中心 z₀ 和函数奇点位置

Step 2: 判断区域类型
        → 圆域 |z-z₀| < R（无奇点）→ 泰勒级数
        → 圆环 r < |z-z₀| < R → 洛朗级数

Step 3: 选择展开方法
        → 直接求导法：aₙ = f⁽ⁿ⁾(z₀)/n!
        → 已知展开式代入（e^z, sin z, 1/(1-z)等）
        → 部分分式分解
        → 代换法

Step 4: 确定收敛半径
        R = |z₀ - z*|（z*为最近奇点）

Step 5: 对洛朗级数：识别负幂项，判断奇点类型
```

## 规则5：留数定理计算实积分（第4章）

**适用问题**：利用留数定理计算实积分

**推理链**：

```
Step 1: 识别积分类型
        类型一：三角有理式 ∫₀²π R(cosθ,sinθ)dθ
                → 令 z = e^(iθ)，dz = izdθ
        类型二：无穷积分 ∫₋∞⁺∞ f(x)dx
                → 补充上半圆围道，Jordan引理
        类型三：含三角函数的无穷积分 ∫₋∞⁺∞ f(x)e^(iax)dx
                → 上半圆/下半圆，Jordan引理

Step 2: 确定辅助函数和围道

Step 3: 找出围道内所有孤立奇点

Step 4: 计算各奇点留数

Step 5: 应用留数定理：∮ = 2πi × ΣRes

Step 6: 证明辅助围道积分趋于0（Jordan引理/大圆弧引理）

Step 7: 取极限，得到实积分值
```

## 规则6：傅里叶/拉普拉斯变换求解（第5-6章，第13章）

**适用问题**：利用积分变换求解微分方程/PDE

**推理链**：

```
Step 1: 选择变换
        → 无界域空间变量 → 傅里叶变换
        → 含初始条件的时间变量 → 拉普拉斯变换

Step 2: 对原方程两边作变换
        微分 → 代数：∂/∂x → iω（傅里叶），∂/∂t → p（拉普拉斯）

Step 3: 解变换后的代数方程，得到像函数

Step 4: 反变换得到原函数
        → 傅里叶：查表/卷积定理
        → 拉普拉斯：留数法/查表/展开定理

Step 5: 验证解满足原方程和定解条件
```

## 规则7：分离变数法求解PDE（第8章）

**适用问题**：求解齐次PDE + 齐次边界条件

**推理链**：

```
Step 1: 分离变量 u(x,t) = X(x)T(t)

Step 2: 代入PDE，分离为两个ODE
        → 空间方程：X'' + λX = 0（本征值问题）
        → 时间方程：T'' + a²λT = 0（或热方程型）

Step 3: 由边界条件求本征值 λₙ 和本征函数 Xₙ(x)

Step 4: 对每个 λₙ 求时间部分解 Tₙ(t)

Step 5: 叠加：u(x,t) = Σₙ Cₙ Xₙ(x) Tₙ(t)

Step 6: 由初始条件定系数 Cₙ（利用正交性）
        Cₙ = ∫φ(x)Xₙ(x)dx / ∫Xₙ²(x)dx
```

## 规则8：特殊函数问题（第9-11章）

**适用问题**：涉及勒让德/贝塞尔函数的计算或证明

**推理链**：

```
Step 1: 识别方程类型
        → (1-x²)y'' - 2xy' + l(l+1)y = 0 → 勒让德方程
        → x²y'' + xy' + (x²-ν²)y = 0 → 贝塞尔方程

Step 2: 识别求解目标
        → 级数展开 → 用Frobenius方法
        → 具体函数值 → 用Rodrigue公式/级数定义
        → 正交展开 → 用正交性定系数
        → 递推计算 → 用递推关系

Step 3: 应用对应性质
        勒让德：正交性 ∫₋₁¹ Pₗ Pₗ' dx = 2/(2l+1) δₗₗ'
        贝塞尔：正交性 ∫₀ᵃ Jᵥ(kₙρ)Jᵥ(kₘρ)ρdρ = 0（n≠m）

Step 4: 结合物理背景验证（量子数、边界条件等）
```

## 规则9：格林函数法求解（第12章）

**适用问题**：求解非齐次PDE/ODE

**推理链**：

```
Step 1: 确定方程和区域
        ∇²u = f(r) 或 u_t = a²∇²u + f(r,t)

Step 2: 选择格林函数构造方法
        → 简单几何 → 电像法
        → 有限区域 → 本征函数展开
        → 无界区域 → 傅里叶变换

Step 3: 构造格林函数 G(r,r')
        满足 ∇²G = δ(r-r') + 边界条件

Step 4: 应用积分公式
        u(r) = ∫ G(r,r')f(r')dV' + 边界项

Step 5: 物理诠释：点源响应的叠加
```

## 规则10：保角变换求解边值问题（第14章）

**适用问题**：求解二维拉普拉斯方程边值问题

**推理链**：

```
Step 1: 分析原区域边界几何形状

Step 2: 选择适当的保角变换 w = f(z)
        → 角域 → 半平面：用 w = zⁿ 或 w = ln z
        → 圆域 ↔ 半平面：用分式线性变换
        → 带域 ↔ 角域：用 w = e^z

Step 3: 验证变换保角性（f'(z) ≠ 0）

Step 4: 将原问题转化为新区域上的简单问题

Step 5: 求解新区域上的边值问题（通常更易）

Step 6: 反变换回原坐标，得到最终解
```

## 通用证明策略（适用各章）

| 待证结论 | 常用工具 |
|----------|----------|
| f(z) = 常数 | Liouville定理、极值原理、C-R方程 |
| f(z) = g(z) | 唯一性定理 |
| |f(z)|的界 | 最大模原理 |
| 解的唯一性 | 能量积分法、极值原理 |
| 解的存在性 | 构造法（格林函数、分离变量） |
| 级数收敛 | 比值法、根值法、比较法 |
| 正交性 | 利用S-L方程自伴性 |
| 完备性 | 引用Sturm-Liouville理论 |

**证明结构模板**：
```
① 明确已知条件
② 明确要证的结论
③ 选择合适的定理/工具
④ 验证定理条件是否满足
⑤ 应用定理得到结论
⑥ （可选）讨论特殊情况/物理意义
```
