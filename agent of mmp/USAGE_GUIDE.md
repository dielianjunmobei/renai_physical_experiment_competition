# USAGE_GUIDE — 数学物理方法助教 Agent 使用说明

---

## 快速开始

### 1. Agent 定位

本 Agent 是《数学物理方法》课程的**全章节AI助教**，覆盖梁昆淼教材全部15章。

**上篇 · 复变函数论（第1-6章）**：
- 复变函数：解析性、C-R方程、初等函数、调和函数
- 复积分：柯西定理、积分公式、高阶导数、Morera定理
- 幂级数：泰勒级数、洛朗级数、收敛性、孤立奇点
- 留数定理：奇点分类、留数计算、实积分应用
- 傅里叶变换：变换对、性质、δ函数、频谱分析
- 拉普拉斯变换：定义、性质、反演、ODE求解

**下篇 · 数学物理方程（第7-15章）**：
- 定解问题：三类方程、定解条件、分类、达朗贝尔公式
- 分离变数法：核心思想、五步骤、本征值问题、叠加原理
- 特殊函数：S-L方程、勒让德/贝塞尔函数、正交性、展开
- 格林函数：点源响应、积分公式、电像法、本征函数展开
- 积分变换法：傅里叶/拉普拉斯变换解PDE
- 保角变换：保角性、常用变换、二维边值问题
- 非线性问题：孤立子、KdV方程、混沌

## 3. 如何调用这个 Agent

### 方式1：直接聊天使用（推荐）

将 `AGENT_CONFIG.md` 中的 "System Prompt" 部分作为系统提示词输入给大模型，然后提问：

```
【系统提示】
（粘贴 AGENT_CONFIG.md 中的 System Prompt 内容）

【用户提问】
用分离变量法求解弦振动方程
```

### 方式2：结构化调用（开发用）

```python
def ask_agent(question: str, chapter: str = None):
    # Step 1: 加载系统提示
    system_prompt = load("AGENT_CONFIG.md", "System Prompt")
    
    # Step 2: 检索相关知识（可选）
    knowledge = retrieve_from("KNOWLEDGE_BASE.md", keywords=extract(question))
    
    # Step 3: 匹配推理规则（可选）
    rule = match_rule("REASONING_RULES.md", question_type=classify(question))
    
    # Step 4: 组合提示并调用模型
    full_prompt = f"{system_prompt}\n\n【知识参考】\n{knowledge}\n\n【推理规则】\n{rule}\n\n【问题】\n{question}"
    
    return model.chat(full_prompt)
```

### 方式3：Streamlit 部署

进入 `deploy/` 目录，运行：

```bash
cd deploy
pip install -r requirements.txt
streamlit run app.py
```

浏览器打开 `http://localhost:8501`，输入 API Key 即可使用。

## 4. 问答格式规范

本 Agent 的每个回答包含6个部分：

```
📌 概念定位：一句话定位知识点（章节+知识点）
📐 核心公式：LaTeX 公式
📝 详细推导：分步推导，每步说明依据
💡 物理直觉：用物理/几何类比解释
⚠️ 常见错误：1-2个典型错误
🎯 自检提问：反问学生，检验理解
```

## 6. 扩展开发建议

1. **知识图谱化**：将 `KNOWLEDGE_BASE.md` 转为 JSON/图数据库，实现知识点关联检索
2. **题目库扩展**：在 `EXAMPLES.md` 基础上增加更多题型，建立标签体系（难度/题型/知识点）
3. **学生模型**：记录学生提问历史，识别薄弱环节（如"C-R方程"总是错→针对性推送练习）
4. **自动出题**：基于知识库生成随机参数的题目
5. **多模态增强**：生成复平面可视化、分离变量法动画、特殊函数图像等
6. **可视化模块**：deploy/visualize.py 已提供复变函数可视化，可扩展为PDE可视化
